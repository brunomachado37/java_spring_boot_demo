# Spring Boot Demo

This is a simple Spring Boot application that demonstrates the basic setup and structure of a Spring Boot project.


# Installation

1. Install the latest version of the Java Development Kit (JDK), if not already installed. You can download it from [here](https://www.oracle.com/java/technologies/downloads/).

2. Install the latest version of Apache Maven, if not already installed. You can follow the instructions from [here](https://maven.apache.org/install.html).

*Note: Recommended installation method for Windows is via [Chocolatey](https://chocolatey.org/install)*

3. Clone this repository to your local machine:
```
git clone https://github.com/brunomachado37/java_spring_boot_demo.git
```

# Running the Application

From the project root directory, run the following command to start the Spring Boot application:
```
mvn spring-boot:run
```

Verify that the application is running by navigating to `http://localhost:8080` in your web browser.
